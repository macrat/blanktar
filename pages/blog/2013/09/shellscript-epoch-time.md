内部的に比較したり何したりするにはやっぱりエポック秒が便利よねー。
というわけで、そのエポック秒を得る方法。

``` shell
$ date +%s
```

はい、以上。簡単。

同じ方法で、
``` shell
$ date "+%Y/%m/%d %T"
2013/09/06 02:15:19
```
みたいなことも可能。詳細は参考のページにあるのでそちらを。

dateコマンドも色々出来るみたいだね―。
